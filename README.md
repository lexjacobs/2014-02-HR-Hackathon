2014-02-HR-Hackathon
====================

February, 2014.
36 hour Hack Reactor solo-hackathon.

data source:
http://bayareabikeshare.com/datachallenge

Rendering tech:
http://d3js.org/

The display auto-advances through each day of January, 2014.
The movement of each dot represents the start/end GPS coordinates of a single bike rental.

In aggregate, the major rental patterns become apparent. Notably, Market Street, 2nd Street, and the Caltrain station.
